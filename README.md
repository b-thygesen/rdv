# Rømø Dragtens Venner

This repository hosts the website for the association **Rømø Dragtens Venner**, published via [GitHub Pages](https://pages.github.com/).

## About the site

The website covers:

- **History** – the story and history of the traditional Rømø dress
- **News & Events** – updates on events and activities the association participates in
- **Gallery** – a photo gallery showcasing the dress and association activities
